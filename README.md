# Narmada Valley Football Club (NVFC)

This repository contains the source code for the Narmada Valley Football Club website and administration system.

## Project Structure

The project is divided into two main directories:

- **`frontend/`**: The Next.js application containing the Website, Admin Panel, and all UI logic.
- **`backend/`**: (Reserved) For future backend services or API code distinct from the Next.js API.

## Getting Started

### Frontend

To work on the website:

1.  Navigate to the frontend directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
